# Ряженка Tuner

Краткое техническое описание

Ряженка Tuner — пакет конфигураций и пресетов для управления производительностью Nintendo Switch под Atmosphere CFW. Набор включает конфигурационные файлы, пресеты для CPU/GPU/RAM и скрипты/файлы обновления.

Назначение

- Предоставить удобную структуру пресетов для быстрого переключения режимов (Handheld, Docked, Energy-Save, Max-FPS).
- Обеспечить совместимость с ядром Ryazhahand и интеграцией sys-clk для точного управления частотами.

Содержание репозитория

- `Ryazhenkabestcfw Tuner/` — основной пакет, содержащий `Data/`, `Tuner/` и `Updater/`.
- `Data/Presets/` — JSON и INI файлы с пресетами для CPU, GPU и RAM.
- `Data/Tuner/` — конфигурации интерфейса тюнера и быстрые действия.

Быстрый старт

1. Сделайте резервную копию текущих профилей и конфигурационных файлов.
2. Скопируйте директорию `Ryazhenkabestcfw Tuner` в место, требуемое вашим CFW/тулом (см. документацию CFW).
3. Перезапустите тюнер или интерфейс, если это требуется для применения новых конфигураций.

Примеры операций

- Применение пресета: выбрать профиль → применить → при необходимости перезапустить интерфейс тюнера.
- Редактирование пресета: отредактировать JSON/INI в `Data/Presets/` и проверить корректность значений перед применением.
